# Quoc Nguyen - Personal Portfolio

A modern, responsive personal portfolio website for Quoc Nguyen, a Software Engineering Leader. Built with Next.js and Tailwind CSS.

## Features

- Clean, modern design with Silicon Valley-inspired UI
- Fully responsive layout
- Smooth animations with Framer Motion
- Interactive components
- Contact form
- PDF resume download
- Optimized performance

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- CSS Grid/Flexbox

## Running Locally

First, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd quoc-portfolio
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

To build the application for production:

```bash
npm run build
```

To start the production server:

```bash
npm start
```

## Customizing

- Modify component content in the `app/components` directory
- Update styles in `app/globals.css` and individual component files
- Replace images in the `public` directory
- Update resume PDF in the `public` directory
